# ParcelVueDemo

[![Greenkeeper badge](https://badges.greenkeeper.io/lovelope/parcel-vue-template.svg)](https://greenkeeper.io/)

## 项目使用

```shell
# 启动项目
npm run dev
```

```shell
# 启动项目
npm run build
```

## plugins

- editorconfig  控制编码和缩进
- eslint        静态js代码校验
- postcss       对更多浏览器做样式兼容
- stylelint     静态css校验
- babel         转译es新特性
